# BlenderCustomPlayStop
A Blender add on which changes blenders playback behaviour to match the one of other video software

## Installation
- Download the file `CustomPlayStop.py`
- Open blender user preferences
- Click `Install...`
- Select the downloaded file
- Enable the add-on

## Usage
For GUI-Button:
- In the 3d-Viewport open the right panel (`N`-Key)
- Click on the `Misc` section
- If closed, open the `Playback` panel
- Use the `Play/Stop`-Button

Keyboard:
- While add on is enabled, press the spacebar to start/stop playback (possible in almost every viewport)
